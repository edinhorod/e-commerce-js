# App

e-Commerce API

## Documentação da API

#### Login

```http
  POST /login
```

| Parâmetro   | Tipo       | Descrição                                   |
| :---------- | :--------- | :------------------------------------------ |
| `email`      | `string` | **Obrigatório**. O e-mail do usuário |
| `password`      | `string` | **Obrigatório**. A senha do usuário |

#### Adicionar Usuário

```http
  POST /users
```

| Parâmetro   | Tipo       | Descrição                                   |
| :---------- | :--------- | :------------------------------------------ |
| `name`      | `string` | **Obrigatório**. O nome do usuário |
| `email`      | `string` | **Obrigatório**. O email do usuário |
| `password`      | `string` | **Obrigatório**. A senha do usuário |
